# CadenceAI — Web Frontend

Next.js 16 + React 19 web client for the CadenceAI voice scheduling assistant.

- **Demo**: `src/app/demo/page.tsx` — Voice agent interface with Pipecat WebRTC
- **Booking**: `src/app/book/[slug]/page.tsx` — Shareable booking pages
- **Components**: `src/components/agent/` — Voice orb, calendar, captions, confirmations
- **Tests**: `npx vitest run`
